从17年的12月到18年4月，大概花了五个月时间读完了《深入理解计算机系统》这本书，也跟着做完了配套的实验，这些实验设计的精妙让我不由得想要花时间去完成它，并享受完成之后的成就感。通过这本书，对CMU这所CS最顶尖的学府的敬佩油然而生，本科生的一个入门课程就能做到如此全面地梳理计算机的系统，让我们清楚地了解一个程序是如何被计算机理解并运行起来的，相比之前我们学过的很多课程往往过于关注知识本身，而忽视了为我们构建一个整体的结构图，脑子里只有一些孤立的知识点，而读了CSAPP之后，这些孤立的知识点才慢慢得连起来，构成了一个完整的计算机系统。这个过程中虽然很多次有过懈怠，想要放弃，但是很庆幸还是坚持了下来。

九章第三讲，宽度优先搜索。主要分为二叉树上的宽搜、图上的宽搜和矩阵上的宽搜三部分。

宽度优先搜索 BFS

什么是宽度优先搜索

先看下wikipedia的解释： Breadth-first search (BFS) is an algorithm for traversing or searching tree or graph. It starts at the tree root or some arbitrary node of a graph, and explores the neighbor nodes first, before moving to the next level neighbours.

通常BFS会需要用到两种数据结构

  1. Queue 利用queue先进先出的特点保证先访问neighbor结点再访问下个level的结点
  2. Set 利用set来记录已经访问过的结点，去重

一般来说BFS分三部分可以实现

  1. 把root(起始结点)放进queue中queue.offer(root);
  2. while循环 while(!queue.isEmpty()){}
  3. for 当前层，拓展出下一层
